📊 Data Transformation Dashboard (Text File to Power BI)

📌 Project Overview

This project demonstrates the transformation of raw text-based data (Notepad format) into a fully interactive Power BI dashboard.

The focus of this project was on data cleaning, structuring, and visualization, turning unorganized data into actionable insights.

📷 Dashboard Preview - https://www.linkedin.com/feed/update/urn:li:activity:7468976706825949184/

🎯 Key Objectives

    •	Convert unstructured text data into a structured dataset 
    
    •	Perform data cleaning and preprocessing 
    
    •	Build an interactive dashboard for analysis 
    
    •	Extract meaningful insights using visualizations 
    

📂 Data Source

    •	Format: .txt (Notepad file) 
    
    •	Nature: Unstructured / Raw data 
    
   •	Steps performed: 
    
    •	Data extraction 
    
    •	Formatting into tabular structure 
    
    •	Handling inconsistencies and missing values
    

🛠️ Tools & Technologies Used

    •	Power BI Desktop – Dashboard development 
    
    •	Power Query – Data cleaning & transformation 
    
    •	DAX – Measures, KPIs, and calculations 
    

📊 Dashboard Highlights

    •	📌 Interactive filters and slicers 
    
    •	📈 Visual insights using charts (bar, line, pie, etc.) 
    
    •	📊 KPI indicators for quick analysis 
    
    •	🎯 User-friendly and clean layout 
    

🔄 Workflow

    1.	Imported raw text data into Power BI 
    
    2.	Cleaned and transformed data using Power Query 
    
    3.	Structured data into meaningful format 
    
    4.	Created calculated columns and measures using DAX 
    
    5.	Designed interactive dashboard visuals 
    

🚀 How to Use

    1.	Download the .pbix file 
    
    2.	Open using Power BI Desktop 
    
    3.	Interact with filters and visuals to explore insights 
    

📈 Key Learnings

    •	Handling raw/unstructured datasets 
    
    •	Data transformation using Power Query 
    
    •	Creating impactful dashboards from basic data 
    
    •	Applying DAX for business insights
